# 5-13

该章节通过一个向量内积的程序，对整数和浮点数类型，在x86-64上CPE等于14~18，通过一系列的优化，例如code motion等，的到一份代码

```cpp
/* inner product. accumulate in temporary */
void inner4(vec_ptr u, vec_ptr v, data_t *dest) {
  long i;
  long length = vec_length(u);
  data_t *udata = get_vec_start(u);
  data_t *vdata = get_vec_start(v);
  data_t sum = (data_t) 0;

  for (i = 0; i < length; i++) {
    sum = sum + udata[i] * vdata[i];
  }
  *dest = sum;
}
```

2. 对于double，这条关键路径决定的CPE的下界是3.0

3. 对于整数代码，关键路径决定的CPE的下界是1.0

4. 虽然乘法操作需要五个时钟周期但是由于流水线的存在，乘法操作的结果可以在第三个时钟周期后被后续的加法操作使用，因此关键路径上乘法的延迟为3个时钟周期，加上加载和加法操作的延迟，总共为3.0个时钟周期。
