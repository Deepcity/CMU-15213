# 5-14

使用 $ 6 \times 1 $ 循环展开，对于这样的版本的测试得到，对整数数据的CPE为1.07，对两种浮点数的CPE仍然为3.01。

Assignments:

1. 为什么在Intel Core i7 Haswell上运行的任何（标量）版本的内积过程都不能达到比1.00更快的CPE?

2. 对浮点数而言为何性能不会通过循环展开而提高

every element has 6 long/float add

element count is n/6

so n/6 * 6 = n
