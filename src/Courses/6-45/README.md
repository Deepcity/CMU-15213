# 6-45

这里要求实现一个尽可能快的矩阵转置函数。

![6-45](https://s2.loli.net/2025/11/09/Sv9CEjTcayd3ebF.png)

## Answer

很容易了解到在矩阵copy类型的执行过程中有两种经典经验：

1. Nn,Zz的不同读写方式转换，这是一种“象形文字”。表达在矩阵copy过程中，Nn的读写方式是按行读取，Zz的读写方式是按列读取。这两种读写方式转换可以在矩阵copy过程中提高效率。并且可以相互嵌套，例如，在小范围行内采取n读取，而在stripe大行内采用Z读写，相反zN也是存在的。

2. 分块读取，在较小缓存的情况下，采用分块的方式尽可能提高缓存利用率。例如，在缓存大小为64字节的情况下，采用8x8的分块读取方式可以最大程度利用缓存。
